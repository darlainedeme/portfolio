Projects

Interactive Dashboard on Energy Access in Africa

This project visualizes the electricity access data across Africa, providing insights into the electrification rates and trends. It includes a choropleth map and bar charts to explore the data interactively.

Data Centers Worldwide

This project maps out data centers across the world, highlighting their distribution and capacity. It aims to provide a comprehensive overview of global data center infrastructure.

Conflict Zones in Africa

This project uses GIS data to map out conflict zones in Africa, providing a visual representation of conflict areas over time. The interactive dashboard helps in understanding the geographical spread and intensity of conflicts.
